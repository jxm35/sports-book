# Sports-Book

- Sports Book is a repo that serves as a place to research and deploy predictive football models.
- In order to support this there are scripts to download historic football results, performances, and odds, and store them to an sql database.
- There is a command to run backtests, this currently supports testing models against the biggest 4 european football leagues, between 2014 and 2023.

It uses the following core tools and technologies

* Go
* Python
* MySql
